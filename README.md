## Excel Sales Dashboard Analysis | SkillCraft Technology Task 01  

---

# Project Overview  

This project was developed as part of **SkillCraft Technology's Data Analytics Internship – Task 01**.  

The objective was to clean, analyze, and visualize the **Sample Superstore Sales Dataset** using **Microsoft Excel** by building a professional interactive sales dashboard with Pivot Tables, KPI cards, slicers, and charts.

---

# Task Objective  

- Clean and prepare raw sales data  
- Analyze total sales and profitability  
- Create Pivot Tables and Pivot Charts  
- Build an interactive dashboard  
- Visualize category and regional performance  
- Track sales trends over time  
- Highlight high and low performers using Conditional Formatting  

---

# Key Business Insights  

- **Technology** generated the highest profits  
- **Furniture** showed the weakest profitability  
- **West Region** performed best overall  
- **Central Region** underperformed  
- Certain sub-categories like **Tables** produced losses  
- Sales trends revealed strong seasonal fluctuations  

---

# Strategic Recommendations  

- Increase investment in profitable Technology products  
- Optimize Furniture pricing and discount strategy  
- Improve Central region sales strategies  
- Focus on high-performing sub-categories  
- Reduce low-profit inventory
