# SCT_DA_1

## Excel Sales Dashboard Analysis

### Task Objective
Analyze Superstore Sales data using Excel by cleaning data, creating Pivot Tables, KPI metrics, charts, and an interactive dashboard.

### Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

### Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Profit Margin KPI
- Profit by Category
- Monthly Sales Trend
- Sub-Category Performance
- Regional Performance
- High/Low Performer Analysis
- Interactive Filters

### Key Insights
- Technology generated highest profit
- West region performed best
- Furniture category had weakest profitability
- Tables sub-category showed losses

### Business Recommendations
- Increase focus on Technology products
- Reduce heavy discounts on Furniture
- Improve Central region strategy
- Optimize underperforming sub-categories

### Files Included
- Excel Dashboard Workbook
- Dataset
- Dashboard Screenshots
- Final Project Report PDF

### Conclusion
This dashboard provides actionable business intelligence through interactive visual analysis and supports strategic decision-making.
